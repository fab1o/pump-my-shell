1. Run the command above to install or upgrade `pump-my-shell`

2. Add `pump` to the plugin list in your Zsh config file: `$HOME/.zshrc`

   ```sh
   plugins=(pump)
   ```

3. Restart your terminal, then run:

   ```sh
   help
   ```
